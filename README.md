# community-announcements
## Description
Community Announcements est une application web collaborative pour publier et consulter les annonces communautaires.

## Technologies utilisées
HTML
CSS
JavaScript
Node.js
Express
MongoDB
Git
## Règles de contribution
'main' : code stable en production
'dev' : développement actif
'feature-*' : branches pour chaque fonctionnalité
toujours créer une **Pull Request (PR)** pour fusionner le code.
Message de commit clairs, en anglais:
exemple: 'git commit -m "Add feature: login page"'
